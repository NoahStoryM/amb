* Introduction

John McCarthy's ambiguous operator for Racket, implemented via
delimited continuations.

This repository contains two packages:

- [[file:pkgs/amb/][amb]] — the core library
- [[file:pkgs/typed-amb/][typed-amb]] — Typed Racket interface

* Quick Start

#+begin_src racket
(require amb)

(for/list ([x (in-amb (amb 1 2 3))])
  x)
;; => '(1 2 3)

(for/list ([t (in-amb (let ([a (for/amb ([i (in-range 1 20)]) i)]
                            [b (for/amb ([i (in-range 1 20)]) i)]
                            [c (for/amb ([i (in-range 1 20)]) i)])
                        (unless (and (<= a b)
                                     (= (+ (* a a) (* b b)) (* c c)))
                          (amb))
                        (list a b c)))])
  t)
;; => '((3 4 5) (5 12 13) (6 8 10) (8 15 17) (9 12 15))
#+end_src

* Installation

#+begin_src sh
raco pkg install amb        # untyped
raco pkg install typed-amb  # typed
#+end_src
